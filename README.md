# 📝 NOTIFY — Minimal Notes Web App

NOTIFY is a clean, minimal, dark-themed notes web app built entirely with pure HTML, CSS and JavaScript. No frameworks, no libraries, no setup required. Just open it in your browser and start writing.

---

## ✨ Features

- 📄 Create notes with a title and content
- 🗂️ Notes displayed as cards in a clean sidebar
- 👁️ Click any note card to read full content
- ✏️ Edit existing notes anytime
- 🗑️ Delete notes with one click
- 💾 Auto saved using localStorage — persists after refresh
- 📊 Live stats showing total notes and total word count
- 🎨 Smooth hover effects and animations throughout

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
- firebase auth and database
- GROQ ai api
